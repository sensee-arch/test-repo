# Test Repo

An experimental project for AI Agent collaborative programming workflow validation.

## Overview

This repository serves as a testing and experimentation ground for:
- AI Agent collaborative programming workflows
- Standardized AI project collaboration processes (requirements → specs → tasks → development → review)
- Reusable AI Agent collaboration templates

## Current Status

Infrastructure setup and collaboration norm establishment phase. See [ARCH.md](ARCH.md) for architecture details.

## Quick Start

```bash
# Clone
git clone https://github.com/sensee-arch/test-repo.git
cd test-repo

# Install Python dependencies (if using backend)
pip install -r requirements.txt

# Install development dependencies
pip install -r requirements-dev.txt
```
